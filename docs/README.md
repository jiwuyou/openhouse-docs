# OpenHouse Official Docs

这是 OpenHouse 的官方文档入口目录。

适用对象：

- 普通用户
- 在 OpenHouse 中运行的 AI
- 为 OpenHouse 配置 skill 或系统提示的人

## 文档顺序

建议按下面顺序阅读：

1. `START_HERE.md`
2. `USER_GUIDE.md`
3. `AGENT_GUIDE.md`
4. `PATHS_AND_PORTS.md`
5. `OFFICIAL_RULES.md`
6. `FAQ.md`
7. `TROUBLESHOOTING.md`
8. `AI_BOOTSTRAP_PROMPT.md`
9. `ENV_SKILL.md`

## 文档性质

本目录中的内容属于 OpenHouse 官方文档。

规则：

- 官方文档优先于 agent 自己的总结
- 官方文档用于定义环境、路径、端口和使用方式
- 如果你需要经验类内容，应单独查看 `agent-notes`

## 环境一句话说明

OpenHouse 运行在基于 Termux 的 Ubuntu 中，AI 应优先读取官方文档，而不是自行猜测系统结构。
